# Logo Images

Place your custom logo images in this folder.

## Required Files

Add the following image files:

- `grokipedia.png` - Logo for Grokipedia/xAI (REQUIRED)
- `britannica.webp` - Logo for Encyclopedia Britannica (REQUIRED)
- `wikipedia.png` (optional) - Logo for Wikipedia (default works well)
- `citizendium.png` (optional) - Logo for Citizendium
- `newworld.png` (optional) - Logo for New World Encyclopedia

## Recommended Dimensions

- Size: 200x200px (will be scaled down in UI)
- Format: PNG, WebP, or SVG
- Background: Transparent preferred

## Currently Configured

The app is configured to use these local logos:
- Grokipedia: `/logos/grokipedia.png`
- Britannica: `/logos/britannica.webp`

And these remote logos (from Wikimedia):
- Wikipedia: Wikipedia puzzle globe
- Citizendium: Citizendium logo
- New World Encyclopedia: Book icon
